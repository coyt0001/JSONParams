## JSON Params ##
 Sending POST parameters in JSON format

## Usage ###
  Launch app

  1. Click: Get Data
     * display the planets; no Pluto
  2. Click: + icon
       * creates a new planet for Pluto
  3. Click: Get Data
       * notice: Pluto
  4. Click: pencil icon
       * updates Pluto to hurdleg
  5. Click: Get Data
       * notice: hurdleg
  6. Click: garbage can icon
       * deletes :planetId 8
  7. Click: Get Data
       * hurdleg the planet is destroyed :O
